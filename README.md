# 🏠 House Price Prediction using Linear Regression

A beginner-friendly Machine Learning project that predicts house prices using **Linear Regression**. This project demonstrates the complete workflow from data loading to model evaluation using Python and Scikit-learn.

---

# 📌 Project Overview

The objective of this project is to predict the selling price of a house based on its characteristics.

This project covers:

- Data Understanding
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Selection
- Train-Test Split
- Feature Scaling
- Linear Regression
- Model Prediction
- Model Evaluation

---

# 📂 Dataset

Target Variable

- **price**

Selected Features

- bedrooms
- bathrooms
- sqft_living
- floors
- grade
- condition
- waterfront
- yr_built
